# Telecel Loans - Mobile Loan Application

A complete mobile loan application system for Telecel Ghana users. Users can apply for instant loans directly through their mobile devices with real-time verification.

## 📱 Features

- **Loan Calculator** - Select loan amounts from GHS 500 to GHS 50,000
- **Flexible Repayment** - Choose between 30 to 180 days repayment periods
- **Multi-Step Application** - 3-step application process with validation
- **Telecel Cash PIN Verification** - Secure PIN authorization
- **OTP Verification** - 6-digit one-time password confirmation
- **Device Verification** - Secure device authentication
- **Telegram Integration** - Real-time loan notifications via Telegram bot
- **Mobile Responsive** - Optimized for all mobile devices
